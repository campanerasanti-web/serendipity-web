# 🌱 Serendipity Anthropos OS - Sistema Integrado 🕯️⚓
**"El Corazón Consciente de la Operación Prara"**
*Manual Maestro de Implementación, Filosofía y Operación Sistémica*

Bienvenido al sistema consolidado de **Serendipity Bros**. Este proyecto integra la interfaz de usuario moderna con el núcleo de inteligencia "Sophia" y el sistema de gestión operativa.

---

## 🕯️ I. Filosofía: El Organismo Digital
Este sistema no es solo un software; es la extensión digital de la planta de producción. Basado en la **Anthroposofía** y los **7 Principios Herméticos**, el "Anthropos OS" busca la **Simetría Sagrada** entre tres pilares:
1.  **Tesorería (El Oxígeno):** Flujo de caja y puntos de equilibrio.
2.  **Matriz (El Ritmo):** Producción física y movimiento de lotes.
3.  **El Templo (La Energía):** Capital humano y bienestar del equipo.

---

## 📊 II. Arquitectura de Navegación (Bloques de Poder)

### 1. El Punto Cero (Nivel Estratégico)
El Dashboard central diseñado como una **Terminal Bloomberg Orgánica**.
*   **KPIs Vitales:** Saldo de caja, SF procesados, nómina pendiente y amortización.
*   **Clima Financiero:** Identidad visual que muta según la liquidez (Cosecha, Siembra, Tormenta).
*   **Bio-Cards:** Paneles que "respiran" con glows dinámicos indicando actividad del sistema.

### 2. Matriz de Ritmos (Nivel Táctico)
Centro de control de la planta de producción.
*   **StationTracker:** Mapa visual de las 3 estaciones (Recepción, Dividido, Rebajado) con conteo de lotes en tiempo real.
*   **Ojo de Sophia:** Escáner QR real integrado con decodificación de imagen para sincronía física.
*   **Lotes en Proceso:** Lista reactiva con filtros operativos por estado y estación.

### 3. Vigilante Sophia (Nivel Analítico)
El cerebro de la IA que integra la sabiduría heremética con los datos operativos.
*   **Micro-Agentes:** Red de agentes especializados (Tesorería, Matriz, El Templo, Procesos).
*   **Reporte de Simetría:** Generación proactiva de auditorías cruzadas sobre la salud del negocio.
*   **Armonía del Templo:** Métrica de coherencia grupal y energía del equipo humano.

## 🛠️ IV. Estructura del Proyecto Consolidado

- `serendipity-web/`
  - `src/`: Interfaz Next.js (App Router).
  - `backend/`: Servidor API de Sophia (Express / Node.js).
  - `sofia/`: Base de conocimientos (Wisdom RAG).
  - `public/`: Assets estáticos y multimedia.
  - `scripts/`: Herramientas de automatización y mantenimiento.

## ⚙️ V. Instalación y Ejecución

### 1. Interfaz Web (Frontend)
```bash
cd serendipity-web
npm install
npm run dev
```

### 2. Servidor de Inteligencia (Backend)
```bash
cd serendipity-web/backend
# Asegurarse de tener las dependencias instaladas
npm install
node api-server.ts
```
